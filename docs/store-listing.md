# Accessibility Snapshot Auditor 商店文案草稿

## English

### Title
Accessibility Snapshot Auditor

### Short Description
Local accessibility snapshots for structure, labels, media, ARIA, keyboard, and contrast.

### Long Description
Accessibility Snapshot Auditor helps developers, designers, QA teams, and web agencies catch common accessibility issues before launch.

Run a local audit on the current page and review grouped findings for:

- Document title, language, H1, heading order, and main landmarks.
- Accessible names for links, buttons, and custom interactive controls.
- Form labels for inputs, selects, and textareas.
- Text alternatives for images and image-like elements.
- ARIA pitfalls such as focusable content hidden from assistive technology.
- Keyboard order issues such as positive tabindex.
- Approximate text contrast checks.

Reports can be exported as HTML or CSV. Audit history and language preferences are stored only in local browser storage. The extension does not upload page content, DOM, screenshots, or audit results.

### Keywords
accessibility audit, a11y checker, WCAG, ARIA, keyboard navigation, contrast checker, browser extension.
