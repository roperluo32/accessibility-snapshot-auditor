# Accessibility Snapshot Auditor 隐私政策草稿

最后更新：2026-04-26

Accessibility Snapshot Auditor 是一个本地无障碍快照审计浏览器插件，用于检查当前网页的结构、可访问名称、表单标签、媒体替代文本、ARIA、键盘顺序和颜色对比。

## 数据处理原则

- 插件默认在本地浏览器中审计当前页面。
- 插件不会默认上传网页正文、DOM、截图或审计结果到服务器。
- 审计历史保存在浏览器本地 storage 中，用户可以在 Options 页面清除。
- 免费功能不需要账号登录。

## 插件会访问的数据

为了完成用户主动触发的审计，插件可能读取当前页面的 title、lang、标题结构、地标元素、链接和按钮文本、图片 alt、表单标签、ARIA 属性、tabindex、可见文本和计算样式。

## 本地保存的数据

插件会在浏览器本地保存最近审计历史和用户选择的界面语言。

## 权限说明

- `activeTab`：用户主动点击审计时访问当前标签页。
- `scripting`：向当前网页注入一次性审计脚本。
- `storage`：保存本地历史和界面语言设置。
- `tabs`：获取当前标签页 ID、URL 和标题，用于审计和历史记录。
