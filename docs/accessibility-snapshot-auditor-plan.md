# Accessibility Snapshot Auditor 免费版开发计划

当前版本定位为完全免费的本地无障碍快照审计插件。

## MVP 范围

- 单页本地审计。
- 文档语言、页面标题、H1、标题层级和 main 地标检查。
- 链接、按钮和自定义交互控件的可访问名称检查。
- 表单 label 检查。
- 图片和 `role="img"` 元素的文本替代检查。
- ARIA hidden focusable、ARIA widget 名称检查。
- 正数 tabindex 检查。
- 基础文本颜色对比检查。
- HTML / CSV 报告导出。
- 本地审计历史。
- 多语言 UI 和 manifest。

## 当前不做

- 远程批量爬取。
- 账号、订阅或支付。
- 法律合规保证。
- 替代人工 WCAG 审核的完整认证报告。
