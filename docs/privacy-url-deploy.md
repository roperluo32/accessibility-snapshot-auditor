# Accessibility Snapshot Auditor 隐私 URL 准备

计划仓库：

- `https://github.com/roperluo32/accessibility-snapshot-auditor`

隐私页面文件：

- `site/privacy.html`

候选公开 URL：

- `https://roperluo32.github.io/accessibility-snapshot-auditor/privacy.html`
- `https://cdn.jsdelivr.net/gh/roperluo32/accessibility-snapshot-auditor@main/site/privacy.html`
- `https://raw.githubusercontent.com/roperluo32/accessibility-snapshot-auditor/main/site/privacy.html`

当前经验：账号级 GitHub Pages 自定义域名可能导致 `github.io` URL 重定向异常。如果 Pages URL 不稳定，优先用 jsDelivr URL 作为商店隐私政策 URL。

发布前必须在无登录窗口中验证 URL 可公开访问。
